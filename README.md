# FdF

This project is about basics of graphic programming, and in particular how to place points in space, how to join them with segments and most importantly how to observe the scene from a particular viewpoint.

```
git clone --recurse-submodules git@github.com:gavrilov/FdF.git
cd FdF
make
```
If you would like to run in a Linux enviroment try to use that library
`https://github.com/ttshivhula/minilibx`
